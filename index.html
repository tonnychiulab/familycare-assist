安全與使用範圍

這是瀏覽器端 BYOK 版本，金鑰無法像後端祕密一樣被完全保護。

  • 未完成機構審查前，請勿輸入可識別的真實病患資料。
  • API Key 會存在此分頁的瀏覽器環境，可能被瀏覽器擴充功能或開發者工具讀取。
  • API Key 不會寫入 GitHub Repository、網址或本網站的 Local Storage。
  • 所有 AI 內容都必須由合格醫事人員覆核。
