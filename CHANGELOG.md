# Changelog

## 1.0.0 — 2026-06-20

- Added the pre-chat setup gate.
- Added clinician System Prompt editor, import, preview, token estimate, and fingerprint.
- Added browser-memory and optional tab-session API Key storage.
- Added Gemini API Key validation through the Models API.
- Added dynamic `generateContent` model selection.
- Replaced the mock responder with live Gemini `generateContent` requests.
- Added runtime handling for invalid keys, quota limits, service errors, safety blocks, and empty replies.
- Added Content Security Policy and GitHub Pages deployment workflow.
- Added five new locale namespaces across six languages.
- Added browser-free unit tests and project consistency checks.
